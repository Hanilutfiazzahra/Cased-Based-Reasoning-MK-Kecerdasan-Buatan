# Cased-Based-Reasoning-MK-Kecerdasan-Buatan

Project ini dibuat untuk menentukan rekomendasi restoran terbaik menggunakan metode Fuzzy Logic berdasarkan dua parameter utama, yaitu:

- Servis
- Harga

Metode fuzzy digunakan untuk menghitung tingkat kelayakan restoran sehingga dapat diperoleh restoran dengan skor terbaik.

---

## Fitur Program

- Membaca data restoran dari file Excel
- Menggunakan fuzzy membership function
- Melakukan fuzzification
- Menggunakan rule inferensi fuzzy
- Menghitung defuzzification dengan weighted average
- Mengurutkan restoran berdasarkan skor fuzzy
- Menampilkan 5 restoran terbaik
- Menyimpan hasil ke file Excel
